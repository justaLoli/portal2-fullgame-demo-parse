Portal 2 Fullgame Demo Summarizer

Drag and drop your demo files here

This is a demo summarizer tool specifically designed for Portal 2 Fullgame runs.
It is based on @nekz/sdp and nekz.me/parser, with modifications for fullgame usage.
Key features include:

  • Groups demos based on splits.
  • Shows time in seconds, which is easier for calculations like min/max, +/-.
  • A single-column table format for easy copy-pasting into spreadsheets for horizontal comparison.

Demos are first sorted by file name, so make sure the demos' names retain their original format (i.e. xxx.dem, xxx_2.dem, ...)

Note: Since the timing is based on demo ticks, it may not be exactly equal to actual split times.
